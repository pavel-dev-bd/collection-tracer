## ğŸ”§ Customization Quick Reference

### Change Primary Color
Edit in `src/css/style.css`:
```css
:root {
  --color-primary-1: #YOUR_COLOR;
}
```

### Enable Dark Mode
System preference is respected automatically. To force:
Edit CSS variables in `@media (prefers-color-scheme: dark)`

### Change Font
Edit in `src/css/style.css`:
```css
body { font-family: 'Your Font', sans-serif; }
```

Full guide: See **CSS_VARIABLES_GUIDE.md**
